## Data Availability

The datasets used in this project are available on Zenodo:

**Zenodo Record:**  
https://zenodo.org/records/20792349

The Zenodo record contains:
- UBE4B benchmark dataset
- GRB2 benchmark dataset
- PTEN activity benchmark dataset
- PTEN abundance benchmark dataset
- Alpha-Amylase benchmark dataset
- DLG4 abundance blind dataset
- DLG4 binding blind dataset
- PETase zero-shot dataset

Original benchmark datasets were obtained from the following repositories:

- https://github.com/gitter-lab/metl-pub
- https://github.com/Align-to-Innovate/the-protein-engineering-tournament-2023
